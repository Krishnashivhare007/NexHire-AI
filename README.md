## ✨ Key Features

- **🧠 Adaptive AI Interviews:** Utilizes advanced GPT models to dynamically generate context-aware interview questions based on candidate responses.
- **🎙️ Real-Time Speech Recognition:** Integrated audio processing and speech-to-text functionality to capture final interview transcripts seamlessly.
- **⚡ Full-Stack Architecture:** Cleanly separated `client` and `server` environments built on the MERN stack for optimal performance and scalability.
- **🔒 Secure & Configured:** Robust backend with advanced CORS configuration to support secure multi-origin requests.
- **📊 Interactive Dashboard:** Intuitive UI for candidates to navigate and finish interviews smoothly.

---

## 🛠️ Tech Stack

**Frontend (`/client`)**
* **React.js:** Core UI library for building dynamic interfaces.
* **Redux Toolkit:** Predictable state management across the application.
* **Tailwind CSS:** Utility-first styling for a responsive and modern design.
* **Web Speech API:** Real-time speech-to-text processing for voice responses.

**Backend (`/server`)**
* **Node.js & Express.js:** Scalable server-side infrastructure and RESTful API routing.
* **MongoDB:** NoSQL database for flexible data modeling of user profiles and interview transcripts.
* **OpenAI API:** LLM integration for generating intelligent, adaptive interview content.
